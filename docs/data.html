Data Preparation

Building a Unified Dataset for Industrial Engineering Programs in Turkey

This page documents the complete data sourcing, cleaning, preprocessing, and integration pipeline used throughout our project.

Where We Got Our Data

To analyze long-term trends in Industrial Engineering education across Turkey, we combined multiple datasets covering the years between 2018 and 2024.

Primary Sources

  • YÖK Atlas Data (2018–2020)
    We used the thestats R package developed by M. Çavuş and O. Aydın to extract official placement statistics directly into R.

  • Kaggle Dataset (2021–2024)
    To enrich the analysis with more recent information, we integrated a Kaggle dataset containing quota sizes, city information, total preference counts, and additional admission statistics.

This hybrid approach allowed us to construct a broader and more consistent analytical timeline.

Data Cleaning and Wrangling

Raw admission datasets often contain inconsistent naming conventions, irrelevant department matches, and formatting issues. To ensure analytical consistency, we applied several preprocessing steps using the tidyverse ecosystem.

Main Cleaning Steps

  • Filtered only Industrial Engineering departments
  • Removed unrelated engineering programs
  • Standardized university and faculty names
  • Fixed Turkish character inconsistencies
  • Created derived analytical variables
  • Grouped universities into State and Foundation categories
  • Combined multiple datasets into a unified structure

Derived Variables

  • Demand_Ratio = Total Preferences / Total Quota
  • Used as a normalized indicator of student demand intensity

Data Dictionary

Below are the most important variables included in our final merged dataset.

Variable Description
Year Academic year between 2018 and 2024
University_Name Standardized university name
University_Type State or Foundation university
Rank National placement ranking
Quota Total student quota
Preferences Total number of student preferences
Demand_Ratio Preferences divided by quota
Top1_Pref Number of first-choice preferences
City City where the university is located

Data Processing Pipeline in R

The following code block contains the complete workflow used to extract, clean, merge, and save the final dataset.

Code 
# Load libraries

library(tidyverse)
library(thestats)

# -----------------------------
# STEP 1 — YÖK Atlas Data
# -----------------------------

raw_thestats <- list_score(
  department_names = "Industrial Engineering",
  lang = "en"
)

clean_thestats <- raw_thestats %>%
  
  filter(as.numeric(year) <= 2020) %>%
  
  filter(
    !str_detect(
      str_to_lower(department),
      "woodworking|fisheries|aquaculture|forest|design"
    )
  ) %>%
  
  mutate(
    University_Type = case_when(
      str_detect(str_to_lower(type), "devlet|state") ~ "State",
      str_detect(str_to_lower(type), "vak|foundation|private") ~ "Foundation",
      TRUE ~ "Other"
    )
  ) %>%
  
  filter(University_Type != "Other") %>%
  
  select(
    Year = year,
    University_Type,
    University_Name = university,
    Faculty_Name = faculty,
    Department_Name = department
  ) %>%
  
  mutate(Year = as.numeric(Year))

# -----------------------------
# STEP 2 — Kaggle Data
# -----------------------------

raw_kaggle <- read_csv(
  "data/01_university_admissions_turkey_2019_2024.csv"
)

clean_kaggle <- raw_kaggle %>%
  
  filter(as.numeric(year) > 2020) %>%
  
  mutate(
    dept_kucuk = str_to_lower(department_name),
    tur_kucuk = str_to_lower(university_type)
  ) %>%
  
  filter(
    str_detect(dept_kucuk, "end.str") &
    str_detect(dept_kucuk, "m.hendis"),
    
    !str_detect(
      dept_kucuk,
      "orman|a.a.|tasar.m|su .r.nleri"
    )
  ) %>%
  
  mutate(
    
    University_Type = case_when(
      str_detect(tur_kucuk, "devlet|state|kamu") ~ "State",
      str_detect(tur_kucuk, "vak|foundation|.zel|private") ~ "Foundation",
      TRUE ~ "Other"
    ),
    
    university_name = str_replace_all(
      university_name,
      "(?i).n.vers.tes.",
      "UNIVERSITY"
    ),
    
    faculty_name = str_replace_all(
      faculty_name,
      "(?i)m.hend.sl.k",
      "Engineering"
    ),
    
    department_name = str_replace_all(
      department_name,
      "(?i)end.str.",
      "Industrial"
    )
  ) %>%
  
  filter(University_Type != "Other") %>%
  
  select(
    Year = year,
    City = city,
    University_Type,
    University_Name = university_name,
    Faculty_Name = faculty_name,
    Department_Name = department_name,
    Rank = final_rank_012,
    Quota = total_quota,
    Preferences = total_preferences,
    Demand_Ratio = demand_per_quota,
    Top1_Pref = top_1_pref_count
  ) %>%
  
  mutate(
    across(
      c(
        Year,
        Rank,
        Quota,
        Preferences,
        Demand_Ratio,
        Top1_Pref
      ),
      as.numeric
    )
  )

# -----------------------------
# STEP 3 — Merge and Save
# -----------------------------

ie_combined <- bind_rows(
  clean_thestats,
  clean_kaggle
) %>%
  
  mutate(
    across(
      c(
        University_Name,
        Faculty_Name,
        Department_Name
      ),
      ~str_to_title(str_squish(.))
    )
  ) %>%
  
  arrange(desc(Year), University_Name)

save(
  ie_combined,
  file = "data/ie_combined_final.RData"
)
