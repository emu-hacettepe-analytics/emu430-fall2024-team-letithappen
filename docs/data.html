Sourcing and Preparation of Industrial Engineering Admission Data

1. Data Overview & Sources

This project utilizes a hybrid data strategy to analyze the trends of Industrial Engineering programs in Turkey between 2018 and 2024. Our objective is to understand the drivers of academic success and regional demand.

We have integrated data from two primary sources:

  • YÖK Atlas (via thestats R package): We utilized the thestats package, developed by M. Çavuş and O. Aydın, to programmatically extract official time-series data (2018-2020) on base scores and success rankings directly from the YÖK Atlas “Lisans Atlası” module.
  • Kaggle Dataset: For the more recent academic years (2021-2024), we integrated a comprehensive Turkish University Admissions dataset to capture advanced metrics such as total preferences, demand-to-quota ratios, and city-based information.

2. Data Preprocessing & Cleaning

Raw data from university admissions is often inconsistent due to naming variations and nested department titles. To ensure a reproducible and high-quality analysis, we implemented the following steps using the tidyverse framework:

  1. Filtering: We isolated Industrial Engineering programs while strictly excluding unrelated fields like Woodworking, Forest, or Fisheries Engineering that appeared in keyword searches.
  2. Harmonization: University and faculty names were standardized (e.g., converting “Üniversitesi” to “UNIVERSITY” and “Mühendislik” to “Engineering”) to allow for seamless merging across different years.
  3. Variable Transformation: We calculated critical metrics such as the Demand Ratio (Total Preferences / Quota) to measure the popularity of programs relative to their capacity.
  4. Type Classification: Institutions were categorized into “State” and “Foundation” sectors to enable comparative benchmarking.

3. Data Dictionary

The following table describes the primary variables used in our final integrated dataset:

Variable Description
Year The academic year of the admission cycle (2018-2024).
University_Name The standardized name of the institution.
Rank The national base placement rank for the IE program (Lower is better).
Demand_Ratio Total number of student preferences divided by the available quota.
Top1_Pref The number of students who placed the program as their first choice.

Data Preparation Code (Internal)

Below is the explicit, reproducible code used to import, clean, and merge the master dataset.

#| label: data-import-preparation #| message: false #| warning: false #| code-fold: true

library(tidyverse) library(thestats)

==========================================

PART 1: YÖK ATLAS API DATA (<= 2020)

==========================================

raw_thestats <- list_score( department_names = “Industrial Engineering”, lang = “en” )

clean_thestats <- raw_thestats %>% filter(as.numeric(year) <= 2020) %>% filter(!str_detect(str_to_lower(department), “woodworking|fisheries|aquaculture|forest|design”)) %>% mutate( University_Type = case_when( str_detect(str_to_lower(type), “devlet|state”) ~ “State”, str_detect(str_to_lower(type), “vak|foundation|private”) ~ “Foundation”, TRUE ~ “Other” ) ) %>% filter(University_Type != “Other”) %>% select( Year = year, University_Type, University_Name = university, Faculty_Name = faculty, Department_Name = department ) %>% mutate(Year = as.numeric(Year))
